# Q-Mart-App
A responsive web app which lists out the products available with a form which saves the product data in to the database.

Database - Postgresql
Framework - Django and Bootstrap

Steps :
1. Run the manage.py file using command : python manage.py runserver and visit the website
2. The website contains 2 web pages which can be visited by clicking on the respective links on Navbar
3. The default first is where you can see the form to add a new product. Fill the form to upload your product.
4. The second page will show you all the products uploaded on the website.
